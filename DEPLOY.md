# Publish This Website

This website is a static site. To share it with others, upload these files to a public web host:

- `index.html`
- `research.html`
- `teaching.html`
- `cv.html`
- `styles.css`
- `assets/`

## Recommended Free Options

### GitHub Pages

1. Create a GitHub repository, for example `meiling-li.github.io`.
2. Upload all files in this folder to the repository.
3. In GitHub, go to Settings > Pages.
4. Choose Deploy from branch, then select `main` and `/root`.
5. Your site will be available at `https://your-github-username.github.io/`.

### Netlify

1. Go to Netlify.
2. Create a new site by uploading this whole folder.
3. Netlify will give you a public URL immediately.

### University Web Space

If the University of Melbourne gives you a personal web folder, upload the same files there.
