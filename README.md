# Personal Academic Website

A simple static personal academic website inspired by a clean faculty/research homepage.

## Edit the content

- Change name, email, roles, institutions, and research interests in `index.html`.
- Add your papers in `research.html`.
- Add teaching and mentoring details in `teaching.html`.
- The CV link points to `assets/CV.pdf`, copied from the updated resume.
- The profile photo is `assets/profile.png`.

## Open locally

Open `index.html` in a browser. No build step is required.
